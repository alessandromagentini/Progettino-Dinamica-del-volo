# Progetto Dinamica del Volo Aerospaziale
### Missione di Intercettazione Asteroide

Progetto universitario per il corso di Dinamica del Volo Aerospaziale.
Sviluppato in MATLAB (e GMAT) da un gruppo di 4 persone.

---

## Obiettivo
Disegnare una missione di intercettazione di un asteroide posizionato su un'orbita 
con inclinazione relativa rispetto al piano equatoriale terrestre.

---

## Contenuti

### [Report 1](https://github.com/alessandromagentini/Progettino-Dinamica-del-volo/blob/main/Documentazione/Report%201.pdf)
- Parametri orbitali
- Posizione del satellite ogni 3 ore
- Disegno dell'orbita in 3 dimensioni (sistema ECI)

### Report 2
- Analisi delle perturbazioni
- Ground track dell'orbita
- Manovre orbitali
- Phasing orbitale
- Disegno delle orbite in 3 dimensioni (sistema ECI)
